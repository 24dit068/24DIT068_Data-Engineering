# Mock Enterprise E-Commerce Dataset Description

## Overview
This project uses a mock enterprise e-commerce dataset to demonstrate the complete data engineering lifecycle.

## Dataset Components

### 1. Transactional Database
Contains structured business data such as:
- Customers
- Orders
- Products
- Payments

### 2. Clickstream JSON Data
Captures customer website activity, including:
- User ID
- Page Views
- Timestamp
- Device Information

### 3. Third-Party Inventory Logs
Contains inventory updates received from external warehouse systems:
- Product ID
- Warehouse
- Stock Level
- Timestamp

## Purpose
The dataset is used to simulate how data flows through the five stages of the data engineering lifecycle:
1. Data Generation
2. Data Ingestion
3. Data Storage
4. Data Transformation
5. Data Serving
